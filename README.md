# Expense Tracker Web App

A simple and responsive Expense Tracker built using **HTML, CSS, and JavaScript**.  
This application allows users to add income and expense transactions, automatically calculate the total balance, and store transaction data using **browser localStorage** so the data remains saved even after refreshing the page.

# Features

- Add income and expense transactions
- Automatic balance calculation
- Transaction history list
- Delete transactions
- Data persistence using localStorage
- Clean and responsive user interface

# Technologies Used

- HTML
- CSS
- JavaScript
